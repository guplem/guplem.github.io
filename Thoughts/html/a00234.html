void Thoughts.ControlSystems.UI.GameUIManager.DisplayUIFor ( MapElement  mapElement,
bool  forceUpdate = false 
)

Displays the UI related to the given MapElement and subscribes the UI to changes on that mapElement that should be reflected.

It also unsubscribes the UI to the changes from the previous map element

Parameters
mapElementThe MapElement from which you want to display the information in the UI.
forceUpdateIgnore if the currently selected MapElement is the same as the new selected MapElement and update the UI as if they were different.

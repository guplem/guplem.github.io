Checks if a MapElement can be spawned or not in a given position.

Parameters
positionCheckThe 2D position to check if a Map
spawningHeightRangeThe minimum height at which the object can be spawned (0 means that can spawn on the sea)
requireNavMeshMust the location require a valid NavMeshSurface?
spawnablePositionIf the given position allows an spawn, this is the 3D position (including the height at which it can be spawned) so there is no need to recalculate it again.
Returns
True if the location allows the spawn of the MapElement, false otherwise.

Transform  parent,
NoiseMapSettings  noiseMapSettings,

Spawns a group of MapElements in the map using a pseudo random perlin noise distribution.

Parameters
objectToSpawnThe map element's prefab to spawn
seedThe seed to use to generate the perlin noise
spawningHeightRangeIn which area of the map this is wanted to spawn. -1 means the bottom of the sea. 1 means the highest points in the world. 0 is the shoreline.
probabilityThe probability of the object being spawned at any given spot (following the perlin noise distribution)
densityThe density of the spawning. 1 meaning all the available spots where the probability says spawn should happen will be filled. 0 means none.
parentThe transform that must be the parent of the spawned MapElement
noiseMapSettingsThe settings to be used for the perlin noise map
requireNavMeshMust the locations where the MapElements will spawn require a valid NavMeshSurface?

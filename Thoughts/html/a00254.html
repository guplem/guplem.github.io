int  valueToCover,
MapElement  executer,

Recursively look for all MapEvents available in the game's map that, as consequence of the event, they make a desired attribute value increase for the owner/executer/target (the needed participant).

Parameters
attributeOwnershipToCoverAttributeOwnership to increase the value of.
valueToCoverThe amount of value needed to be covered (increased).
executerMap element that is going to execute the list of ExecutionPlans.
mapEventsToExecuteExecution plans wanted to be executed previously to the ones to cover the attributeToCover.
iterationThe iteration number of the this method's recursive execution. Should start as 0.
Returns
An ordered list of the Execution Plans needed to achieve the goal (to increase the value of the attributeToCover by valueToCover)
